# Этап 0: Git

## Цель

Создать отдельную ветку для разработки функционала Docker-изоляции сабагентов.

## Действия

```bash
git checkout -b feature/docker-subagent-isolation
```

## Результат

- Создана ветка `feature/docker-subagent-isolation` от main/master
- Все изменения будут вестись в этой ветке
